# Отчет по исследованию продаж телефонов

## Описание решения
Для анализа популярности цветов был составлен SQL-запрос, объединяющий таблицы покупок (`table_checkout`) и характеристик устройств (`table_phones`) по идентификатору `phone_id`. 

Использование агрегирующей функции `COUNT(*)` на стороне СУБД позволило выполнить расчет максимально быстро, следуя рекомендациям по производительности (расчет в SQL эффективнее обработки данных в Python).

## Результаты анализа
1. **Телефоны какого цвета чаще всего покупают?**
   Самый популярный цвет — **синий**. Всего зафиксировано **500** продаж.

2. **Какие телефоны чаще покупают: красные или синие?**
   - Синие: 500 продаж.
   - Красные: 429 продаж.
   **Ответ**: Синие телефоны покупают чаще.

3. **Какой самый непопулярный цвет телефона?**
   Самым непопулярным оказался **золотой** цвет, на него пришлось всего **28** продаж.
